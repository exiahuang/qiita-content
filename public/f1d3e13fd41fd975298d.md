# Auto Create VF-Controller-DTO-DAO-Code

> Tips: It just create the code in the localhost. It will not deploy to your sfdc server.

## Find the menu 

SFDC Code Creator > Create VisualForce/Controller/DTO/DAO Code

![1540444254303](http://salesforcexytools.com/images/xytools_images/1540444254303.png)

## Choose your Sobject.

I will select `Blog__c`.

![1540444395082](http://salesforcexytools.com/images/xytools_images/1540444395082.png)

About `Blog__c`:

![1540535859753](http://salesforcexytools.com/images/xytools_images/1540535859753.png)





## Select Custom Fields Or All Fields.

I will select `1.Custom fields Only(Exculde Validate)`.

![1540444454039](http://salesforcexytools.com/images/xytools_images/1540444454039.png)



When It is done , It will show the popup like below.

There are many source will be opened if you choose `ok`. 

![1540444924569](http://salesforcexytools.com/images/xytools_images/1540444924569.png)

You can find the source in `./src_sfdc_module/code-creator/src`

![1540445085932](http://salesforcexytools.com/images/xytools_images/1540445085932.png)



# Deploy the codes

choose the `src`, then right click, `Sfdc-Xy > Deploy Directory To SFDC`

![1540445304166](http://salesforcexytools.com/images/xytools_images/1540445304166.png)



> Tips : If you build a complex page , maybe you can not deploy correctly. Fix the codes, then deploy them.
